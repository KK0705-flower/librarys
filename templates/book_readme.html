このアプリの使い方

「KaienBOOKサーチ」はKaien社内にある蔵書を登録、貸出、返却が出来るアプリです。

TOP画面

  1. データベースに登録された本が表示されます。
  2. 蔵書を登録するページに移動します。
  3. 登録した蔵書を検索、貸出するページに移動します。
  4. 貸出中の蔵書を確認し、返却するページに移動します。
  5. 当ページに移動するリンクです。

蔵書の登録

  1. ISBNコードを入力： フォームの①の部分に、本の背面にある10桁または13桁のコードを入力します。
  2. 情報を取得ボタンを押す： ②のボタンをクリックすると、本のタイトルや著者名が自動で取得されます。

蔵書の検索と貸出

  1. キーワード入力： ①の検索ボックスに、探したい本のタイトルや著者名を入力します。
  2. 検索条件の指定： ②でお好みの検索スタイルで本を検索します。
  3. 検索実行： ③の「この条件で検索する」ボタンをクリックします。
  4. 本の確認： ④に表示された検索結果から、借りたい本の情報を確認します。
  5. 返却予定日の入力： ⑤のカレンダーマークをクリックして、本を返す日を指定します。
  6. 貸出完了： ⑥の大きな赤い「この本を借りる」ボタンを押せば手続き完了です！

蔵書の返却

  1. 借りている本の確認： ①の「現在借りている本」の一覧から、返却したい本のタイトルと返却期限を確認します。
  2. 返却処理： 本をもとの本棚に戻したら、本の右側にある「返却する」ボタンをクリックします。

    ※期限を過ぎないようご注意ください！

  3. 完了： 一覧から本が消えた場合、返却したことになります。
